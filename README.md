
### Repository created for programming and development in the Scala language.
![Scala](https://img.shields.io/badge/Scala-F80000?style=for-the-badge&logo=scala&logoColor=white) 

Scala is a programming language that is often used in conjunction with the Apache Spark ecosystem, making it a solid choice for data professionals in various situations.
This repository was created to guide my learning in Scala and its use in data engineering.
To discover how to manipulate and process large volumes of data efficiently using the powerful resources offered in Scala

Scala é uma linguagem de programação que é frequentemente usada em conjunto com o ecossistema do Apache Spark, o que a torna uma escolha sólida para profissionais de dados em várias situações. 
Este repositório foi criado para guiar meu aprendizado em Scala e sua utilização na engenharia de dados. 
Para descubrir como manipular e processar grandes volumes de dados de forma eficiente usando os recursos poderosos oferecidos em Scala.
